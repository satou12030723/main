# Linux基本コマンド一覧

## ファイル・ディレクトリ操作

- **ls** - ファイルやディレクトリの一覧を表示
  - `ls -l` 詳細情報を表示
  - `ls -a` 隠しファイルを含めて表示
  - `ls -lh` 人間が読みやすい形式でサイズを表示

- **cd** - ディレクトリを移動
  - `cd /path/to/dir` 指定したディレクトリへ移動
  - `cd ..` 一つ上のディレクトリへ移動
  - `cd ~` ホームディレクトリへ移動
  - `cd -` 直前にいたディレクトリへ移動

- **pwd** - 現在のディレクトリのフルパスを表示

- **mkdir** - 新しいディレクトリを作成
  - `mkdir dirname` ディレクトリを作成
  - `mkdir -p dir1/dir2/dir3` 親ディレクトリも含めて一括作成

- **rm** - ファイルやディレクトリを削除
  - `rm filename` ファイルを削除
  - `rm -r dirname` ディレクトリを再帰的に削除
  - `rm -f filename` 確認なしで強制削除
  - `rm -rf dirname` ディレクトリを確認なしで強制削除

- **cp** - ファイルやディレクトリをコピー
  - `cp file1 file2` file1をfile2としてコピー
  - `cp -r dir1 dir2` ディレクトリを再帰的にコピー
  - `cp file1 /path/to/dir/` ファイルを指定ディレクトリにコピー

- **mv** - ファイルやディレクトリを移動・名前変更
  - `mv oldname newname` 名前を変更
  - `mv file /path/to/dir/` ファイルを移動

- **touch** - 空のファイルを作成、またはタイムスタンプを更新
  - `touch filename` 新しいファイルを作成

- **find** - ファイルやディレクトリを検索
  - `find /path -name "filename"` 名前で検索
  - `find . -type f` カレントディレクトリ以下のファイルを検索
  - `find . -type d` ディレクトリのみを検索

## ファイル内容の表示・編集

- **cat** - ファイルの内容を表示
  - `cat filename` ファイルの内容を全て表示
  - `cat file1 file2` 複数ファイルを連結して表示

- **less** - ファイルの内容をページ単位で表示
  - `less filename` スクロール可能な形式で表示
  - スペースキーで次ページ、qで終了

- **more** - ファイルの内容をページ単位で表示（lessより機能が少ない）

- **head** - ファイルの先頭部分を表示
  - `head filename` デフォルトで先頭10行を表示
  - `head -n 20 filename` 先頭20行を表示

- **tail** - ファイルの末尾部分を表示
  - `tail filename` デフォルトで末尾10行を表示
  - `tail -n 20 filename` 末尾20行を表示
  - `tail -f filename` ファイルの更新をリアルタイムで監視

- **nano** - シンプルなテキストエディタ
  - `nano filename` ファイルを編集

- **vi/vim** - 高機能なテキストエディタ
  - `vi filename` ファイルを編集
  - iキーで入力モード、Escで通常モード、:wqで保存して終了

## ファイル検索・テキスト処理

- **grep** - ファイル内のテキストを検索
  - `grep "pattern" filename` パターンに一致する行を表示
  - `grep -r "pattern" /path` ディレクトリ内を再帰的に検索
  - `grep -i "pattern" filename` 大文字小文字を区別せずに検索
  - `grep -n "pattern" filename` 行番号付きで表示

- **wc** - ファイルの行数、単語数、バイト数をカウント
  - `wc filename` 行数、単語数、バイト数を表示
  - `wc -l filename` 行数のみ表示

- **sort** - テキストを行単位でソート
  - `sort filename` アルファベット順にソート
  - `sort -r filename` 逆順にソート
  - `sort -n filename` 数値としてソート

- **uniq** - 重複した行を削除
  - `uniq filename` 連続する重複行を削除
  - `sort filename | uniq` ソート後に重複を削除

## システム情報・プロセス管理

- **ps** - 実行中のプロセスを表示
  - `ps` 現在のシェルのプロセスを表示
  - `ps aux` 全ユーザーの全プロセスを詳細表示

- **top** - プロセスをリアルタイムで表示・監視
  - `top` CPU使用率順にプロセスを表示
  - qキーで終了

- **htop** - topの改良版（インストールが必要な場合あり）

- **kill** - プロセスを終了
  - `kill PID` プロセスIDを指定して終了
  - `kill -9 PID` 強制終了

- **df** - ディスク使用状況を表示
  - `df -h` 人間が読みやすい形式で表示

- **du** - ディレクトリやファイルのサイズを表示
  - `du -h` 人間が読みやすい形式で表示
  - `du -sh dirname` ディレクトリの合計サイズを表示

- **free** - メモリ使用状況を表示
  - `free -h` 人間が読みやすい形式で表示

- **uname** - システム情報を表示
  - `uname -a` 全てのシステム情報を表示

- **hostname** - ホスト名を表示

- **uptime** - システムの稼働時間を表示

## ネットワーク関連

- **ping** - ネットワーク接続を確認
  - `ping google.com` 指定ホストへの接続を確認
  - Ctrl+Cで停止

- **wget** - ファイルをダウンロード
  - `wget URL` 指定URLのファイルをダウンロード

- **curl** - データを転送（HTTP、FTPなど）
  - `curl URL` URLの内容を表示
  - `curl -O URL` ファイルとしてダウンロード

- **ssh** - リモートホストに安全に接続
  - `ssh user@hostname` リモートホストにログイン

- **scp** - ファイルをリモートホストとの間で安全にコピー
  - `scp file user@host:/path` ローカルからリモートへコピー
  - `scp user@host:/path/file .` リモートからローカルへコピー

- **ifconfig** または **ip** - ネットワークインターフェースの設定を表示
  - `ifconfig` ネットワーク設定を表示
  - `ip addr` ネットワーク設定を表示（新しい方法）

## 権限・所有者管理

- **chmod** - ファイルやディレクトリのアクセス権限を変更
  - `chmod 755 filename` 権限を数値で指定
  - `chmod +x filename` 実行権限を追加
  - `chmod -w filename` 書き込み権限を削除

- **chown** - ファイルやディレクトリの所有者を変更
  - `chown user filename` 所有者を変更
  - `chown user:group filename` 所有者とグループを変更
  - `chown -R user dirname` ディレクトリを再帰的に変更

- **sudo** - 管理者権限でコマンドを実行
  - `sudo command` スーパーユーザー権限でコマンド実行

## 圧縮・解凍

- **tar** - ファイルをアーカイブ
  - `tar -cvf archive.tar files` ファイルをアーカイブ
  - `tar -xvf archive.tar` アーカイブを展開
  - `tar -czvf archive.tar.gz files` gzip圧縮してアーカイブ
  - `tar -xzvf archive.tar.gz` gzip圧縮アーカイブを展開

- **gzip** - ファイルを圧縮
  - `gzip filename` ファイルをgzip圧縮
  - `gzip -d filename.gz` 解凍

- **zip/unzip** - ZIP形式で圧縮・解凍
  - `zip archive.zip files` ファイルをZIP圧縮
  - `unzip archive.zip` ZIP解凍

## その他便利なコマンド

- **man** - コマンドのマニュアルを表示
  - `man command` コマンドの使い方を表示
  表示したリストを閉じる - ctrl + z

- **history** - コマンド履歴を表示
  - `history` 過去に実行したコマンドを表示

- **clear** - ターミナル画面をクリア

- **echo** - テキストを表示
  - `echo "text"` テキストを出力
  - `echo $VARIABLE` 環境変数の値を表示

- **date** - 現在の日時を表示

- **whoami** - 現在のユーザー名を表示

- **which** - コマンドのパスを表示
  - `which command` コマンドの実行ファイルの場所を表示

- **alias** - コマンドのエイリアスを設定
  - `alias ll='ls -la'` エイリアスを作成

## パイプとリダイレクト

- **|** (パイプ) - コマンドの出力を別のコマンドへ渡す
  - `command1 | command2` command1の出力をcommand2の入力に

- **>** - 出力をファイルにリダイレクト（上書き）
  - `command > file` コマンドの出力をファイルに保存

- **>>** - 出力をファイルに追記
  - `command >> file` コマンドの出力をファイルに追記

- **<** - ファイルから入力
  - `command < file` ファイルの内容をコマンドの入力に
